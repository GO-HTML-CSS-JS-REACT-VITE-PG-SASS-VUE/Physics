Изучаем Blender 3.4

№ Задание Описание и пошаговая инструкция
1 Создание простого куба
Откройте Blender 3.4
Удалите стандартный куб (X → Delete)
Добавьте новый куб: Shift+A → Mesh → Cube
Масштабируйте куб клавишей S, затем введите 2
Примените цвет: выберите Material Properties → New → измените Base Color
Легко
2 Моделирование стола
Добавьте куб для столешницы: Shift+A → Mesh → Cube
Масштабируйте по осям: S → Z → 0.1 (плоская столешница)
Затем S → X → 2, S → Y → 1.5
Добавьте 4 цилиндра для ножек: Shift+A → Mesh → Cylinder
Масштабируйте ножки: S → X,Y → 0.1, затем S → Z → 1.5
Переместите ножки в углы стола (клавиша G)
Легко
3 Освещение сцены
Выберите стандартный источник света
Переместите его: G → Z → 5
Измените тип света: Light Properties → Type → Area
Увеличьте мощность: Power → 500W
Добавьте второй свет: Shift+A → Light → Point Light
Настройте цвет света в Light Properties → Color
Легко
4 Создание материала с текстурой
Выберите объект
Перейдите в Shading workspace (вкладка вверху)
Нажмите New в Material Properties
В Shader Editor: Shift+A → Texture → Image Texture
Соедините Color из Image Texture с Base Color в Principled BSDF
Загрузите изображение через Open в ноде Image Texture
Средне
5 Модификатор Subdivision Surface
Создайте куб: Shift+A → Mesh → Cube
Выберите куб и перейдите в Modifier Properties (значок гаечного ключа)
Add Modifier → Subdivision Surface
Увеличьте Levels Viewport до 3
Добавьте модификатор Bevel для сглаживания краёв
Примените модификатор: правый клик → Apply
Легко
6 Создание простой чашки
Добавьте цилиндр: Shift+A → Mesh → Cylinder
Tab (Edit Mode) → выберите верхнюю грань
I (Inset) → двигайте мышью внутрь → клик
E (Extrude) → двигайте вниз → клик
Добавьте модификатор Solidify для толщины стенок
Добавьте ручку из тора: Shift+A → Mesh → Torus
Средне
7 Анимация вращения
Выберите объект для анимации
Перейдите на кадр 1 (Timeline внизу)
Нажмите I → Rotation для создания ключевого кадра
Перейдите на кадр 120
Поверните объект: R → Z → 360
Снова I → Rotation для второго ключевого кадра
Нажмите пробел для просмотра анимации
Легко
8 Создание стеклянного материала
Выберите объект и перейдите в Shading
В Principled BSDF установите Transmission = 1.0
Roughness = 0.0 для идеального стекла
IOR (Index of Refraction) = 1.45
Включите рендер Cycles: Render Properties → Cycles
Переключите Viewport Shading на режим рендера (Z → 8)
Средне
9 Моделирование подушки
Создайте куб: Shift+A → Mesh → Cube
Масштабируйте: S → Z → 0.3
Добавьте Subdivision Surface модификатор (2-3 уровня)
Tab (Edit Mode) → выберите все вершины (A)
Alt+S для масштабирования по нормалям → уменьшите
Добавьте Cloth Simulation для мягкости (Physics Properties)
Средне
10 Создание камеры и рендер
Выберите камеру в сцене
Нажмите Numpad 0 для вида из камеры
G для перемещения камеры, R для поворота
Ctrl+Alt+Numpad 0 для привязки камеры к текущему виду
Настройте разрешение: Output Properties → Resolution
F12 для рендера одного кадра
Image → Save As для сохранения
Легко
11 Модификатор Array
Создайте объект (например, куб)
Add Modifier → Array
Увеличьте Count до 10
Измените Relative Offset по осям X, Y, Z
Добавьте второй Array модификатор для сетки
Комбинируйте с Curve для создания рядов вдоль пути
Средне
12 Создание эмиссивного материала
Выберите объект, перейдите в Shading
В Principled BSDF найдите параметр Emission
Установите Emission Strength = 5.0 или выше
Выберите цвет свечения в Emission Color
Включите Bloom в Render Properties → Bloom
Переключите в режим рендера для просмотра
Легко
13 UV-развертка объекта
Выберите объект и перейдите в Edit Mode (Tab)
Выберите все грани (A)
U → Unwrap (или Smart UV Project)
Откройте UV Editor в другом окне
Настройте развертку, перемещая UV-острова
Экспортируйте UV Layout: UV → Export UV Layout
Средне
14 Создание лестницы
Создайте куб для одной ступени
Масштабируйте: S → X → 2, S → Z → 0.2
Добавьте Array модификатор
Count = 10, Relative Offset: Z = 1.5, Y = -1
Создайте перила из цилиндров с Array
Объедините элементы: Ctrl+J
Средне
15 Particle System - трава
Создайте плоскость: Shift+A → Mesh → Plane
Масштабируйте: S → 10
Перейдите в Particle Properties → New
Выберите Hair, Number = 5000
В Render измените Render As на Object
Создайте травинку из куба и выберите её как Instance Object
Сложно
16 Модификатор Mirror
Создайте половину объекта (например, половину вазы)
Добавьте модификатор Mirror
Выберите ось отражения (X, Y или Z)
Включите Clipping для слияния центральных вершин
В Edit Mode редактируйте одну сторону
Примените модификатор после завершения
Легко
17 Создание металлического материала
Перейдите в Shading workspace
В Principled BSDF: Metallic = 1.0
Roughness = 0.2-0.4 для полированного металла
Base Color → выберите цвет металла
Добавьте HDRI освещение: World Properties → Color → Environment Texture
Загрузите HDRI карту для реалистичных отражений
Средне
18 Sculpting - простая голова
Добавьте UV Sphere: Shift+A → Mesh → UV Sphere
Перейдите в Sculpting workspace
Используйте Draw brush для добавления объема
Grab brush (G) для перемещения областей
Smooth brush (S) для сглаживания
Создайте нос, глаза, рот базовыми формами
Используйте Remesh для увеличения детализации
Сложно
19 Создание пончика (Donut)
Добавьте Torus: Shift+A → Mesh → Torus
Масштабируйте по необходимости
Tab → выберите верхние грани
Добавьте Subdivision Surface модификатор
Создайте материал глазури с Glossy shader
Добавьте посыпку через Particle System
Средне
20 Geometry Nodes - простой узел
Выберите объект
Перейдите в Geometry Nodes workspace
Нажмите New в Geometry Nodes Editor
Shift+A → Mesh → Mesh to Points
Shift+A → Instances → Instance on Points
Создайте объект для инстансирования (например, куб)
Соедините ноды: Group Input → Mesh to Points → Instance on Points → Group Output
Сложно
21 Риггинг простого персонажа
Создайте простую модель человечка из кубов
Shift+A → Armature → Single Bone
Edit Mode (Tab) → E для экструдирования костей
Создайте скелет: позвоночник, руки, ноги
Выберите меш, затем арматуру: Ctrl+P → Automatic Weights
Pose Mode → тестируйте движения
Сложно
22 Создание огня с симуляцией
Создайте куб: Shift+A → Mesh → Cube
Добавьте второй куб большего размера (домен)
Выберите малый куб: Physics Properties → Fluid → Flow
Flow Type → Fire
Выберите большой куб: Physics Properties → Fluid → Domain
Domain Type → Gas, нажмите Bake All
Анимация огня готова!
Сложно
23 Создание комнаты
Создайте куб и масштабируйте: S → 5
Edit Mode → выберите одну грань → Delete → Faces
Alt+E → Extrude Faces Along Normals внутрь
Добавьте окна: выберите грань → I (Inset) → E → Delete
Добавьте пол, мебель из примитивов
Настройте освещение через окна
Средне
24 Cloth Simulation - скатерть
Создайте плоскость: Shift+A → Mesh → Plane
Subdivision Surface → 5-6 уровней для детализации
Physics Properties → Cloth → активируйте
Создайте стол под скатертью
Для стола: Physics Properties → Collision
Запустите анимацию (пробел) для симуляции падения
Средне
25 Volumetric Lighting
Создайте сцену с объектами
Shift+A → Light → Spot Light
Направьте свет на объекты
World Properties → Volume → Add Volume Scatter
Density = 0.01-0.05 для видимых лучей
Render Properties → Cycles → увеличьте Samples
Средне
26 Создание планеты
Добавьте UV Sphere: Shift+A → Mesh → UV Sphere
Subdivision Surface модификатор
Shading → добавьте Image Texture с картой планеты
Добавьте Bump/Normal map для рельефа
Создайте атмосферу из большей сферы с прозрачным материалом
Добавьте вращение через анимацию
Сложно
27 Модификатор Boolean
Создайте два пересекающихся объекта (кубы)
Выберите первый объект
Add Modifier → Boolean
Operation → Difference (или Union, Intersect)
Object → выберите второй объект
Используйте для создания отверстий, вырезов
Средне
28 Создание дерева
Создайте цилиндр для ствола
Edit Mode → выберите верхнюю грань
E → экструдируйте несколько раз, масштабируя (S)
Для кроны: добавьте Ico Sphere
Subdivision Surface для сглаживания
Используйте Proportional Editing (O) для органичной формы
Создайте зеленый материал для листвы
Средне
29 Compositing - цветокоррекция
Создайте и отрендерите сцену (F12)
Перейдите в Compositing workspace
Включите Use Nodes
Shift+A → Color → RGB Curves (настройка контраста)
Shift+A → Color → Hue/Saturation (насыщенность)
Shift+A → Filter → Glare (эффект свечения)
Соедините ноды: Render Layers → эффекты → Composite
Сложно
30 Финальная сцена - натюрморт
Создайте стол (задание #2)
Разместите несколько объектов: чашка, фрукты, книги
Настройте камеру с интересным углом (правило третей)
Добавьте 3-точечное освещение: Key, Fill, Rim
Создайте реалистичные материалы для каждого объекта
Включите Ambient Occlusion в Render Properties
Настройте Depth of Field в камере
Отрендерите финальное изображение высокого качества
Сложно
