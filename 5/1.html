Движение тела, брошенного под углом к горизонту

Основные уравнения движения

Ускорение:

$$\vec{a} = \vec{g}$$

Перемещение:

$$\vec{s} = \vec{v}_0 t + \frac{\vec{g}t^2}{2}$$

Скорость:

$$\vec{v} = \vec{v}_0 + \vec{g}t$$

Дано: $v_0$, $\alpha$

Основные характеристики движения

1. Время полета $t_n = ?$

$$t_n = \frac{2v_0 \sin\alpha}{g}$$

2. Максимальная высота подъема $h_{max} = ?$

$$h_{max} = \frac{v_0^2 \sin^2\alpha}{2g}$$

3. Дальность полета $L = ?$

$$L = \frac{v_0^2 \sin 2\alpha}{g}$$

4. Уравнение траектории $y(x) = ?$

$$y = x \cdot \tan\alpha - \frac{gx^2}{2v_0^2 \cos^2\alpha}$$

Составляющие начальной скорости

Горизонтальная составляющая:

$$v_{0x} = v_0 \cos\alpha$$

Вертикальная составляющая:

$$v_{0y} = v_0 \sin\alpha$$

Скорость в произвольный момент времени

Горизонтальная скорость (постоянна):

$$v_x = v_0 \cos\alpha$$

Вертикальная скорость:

$$v_y = v_0 \sin\alpha - gt$$

Модуль полной скорости:

$$v = \sqrt{v_x^2 + v_y^2}$$
